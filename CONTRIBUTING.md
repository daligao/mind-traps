# Contributing to Mind Traps

## How to Contribute

### 1. Suggest New Scenarios

Found an interesting real-world scenario that illustrates a psychology law?

1. Open an [Issue](https://github.com/daligao/mind-traps/issues)
2. Include:
   - Which law it illustrates
   - Scenario description
   - Why it's a good example

### 2. Report Issues

- Bug in the game?
- Wrong answer marked as correct?
- Typo in explanation?
- Scenario doesn't match the law?

Open an Issue with:
- What went wrong
- Which scenario/law
- Screenshot if applicable

### 3. Improve Explanations

- Is an explanation unclear?
- Better way to phrase something?
- Have a better real-world example?

Submit a PR with your improvement.

### 4. Translate

Want to translate to another language? Let's do it!

1. Open an Issue to discuss
2. Create translated version
3. Submit PR

## Code Guidelines

- Keep it simple — no external dependencies
- Maintain mobile compatibility
- Test on multiple browsers
- Follow existing code style

## Questions?

Open an Issue with the "question" label.

**Thank you for contributing!**
